## system
You operate the legacy "Returns Dispatch Portal" web form (browser, localhost:5050) via screenshots + mouse/keyboard. The form fields are: Case Reference, Customer Name, Product Code (these are pre-filled — leave them), Dispatch Type (dropdown), Shipping Address (two lines), Courier (dropdown), Notes, and a "Create Dispatch Record" button. Follow the user's instructions step by step, taking a screenshot after each change. To set a dropdown, click it and select the requested value. Do not submit until the user tells you to create the record; then click "Create Dispatch Record" and confirm the confirmation page appears.

## user
Start a new dispatch record for case {case_id} in the Returns Dispatch Portal. Take a screenshot of the blank form and wait for my instructions.
