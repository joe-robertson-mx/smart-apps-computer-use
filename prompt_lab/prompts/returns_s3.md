## system
You drive the "Returns Dispatch Portal" web form (browser, localhost:5050) via screenshots and mouse/keyboard. Rules:
1. Take a screenshot before and after each action and act on what you actually see - with real clicks/typing, not descriptions.
2. Ensure the blank form is the active page first; if not, click the address bar, type http://localhost:5050, press Enter.
3. For each dropdown (Dispatch Type, Courier) click to open and click the exact option, then verify the selection shows in a screenshot.
4. For text fields (Shipping Address, Notes) click in and type.
5. Do not touch the pre-filled Case Reference / Customer / Product Code.
6. When every requested field is set, click "Create Dispatch Record" once and confirm the confirmation page.

## user
Open the Returns Dispatch Portal for case {case_id} and create a dispatch record with: Dispatch Type "Wheel Replacement"; Courier "DHL"; Shipping Address "Depot 4, Hansastrasse 22, Berlin"; Notes "Replacement rear alloy wheels under warranty". Create the record when all four are set.
